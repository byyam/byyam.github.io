Jekyll

特点：官方推荐，Ruby环境

创建仓库

根据官方文档,

https://pages.github.com/

创建一个同名的仓库。

Jekyll的使用

On Ubuntu,

$sudo apt-get install ruby ruby-dev nodejs gem
$sudo gem install jekyll

On Mac,

$brew install ruby
$gem install jekyll

创建blog,

$jekyll new myblog
$cd myblod
$jekyll serve

可以在这里换主题,

http://jekyllthemes.org/

_posts 就是放Markdown文章的目录,

$mkdir _posts

文件名格式： yyyy-mm-dd-postname.md
